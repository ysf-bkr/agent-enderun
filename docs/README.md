Agent Enderun is designed as an **Autonomous Army Command System**. Unlike traditional frameworks, it focuses on:

1.  **Constitutional Discipline**: AI agents are bound by code-level rules and Markdown-based laws.

3.  **Cross-Platform Synergy**: Unified orchestration across Gemini, Claude, Grok, and more.
